    # Glassbox 2.0

    Open-source mechanistic interpretability. Attribution patching, minimum faithful
    circuits, bootstrap confidence intervals, cross-model alignment.

    ## Install
```bash
    pip install glassbox
```

    ## CLI
```bash
    glassbox analyze \
      --prompt "When Mary and John went to the store, John gave a gift to" \
      --correct " Mary" --incorrect " John"
```
